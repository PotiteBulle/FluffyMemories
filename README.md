# FluffyMemories
FluffyMemories : Vider automatiquement la mémoire Standby sur Windows


Comment utilise le script: 

1 - Télécharge le dossier Git complet avec le fichier `EmptyStandbyList.exe`

2 - Place-le fichier `EmptyStandbyList.exe` dans `C:\Windows\System32\`

3 - Exécute le fichier en mode administrateurice
